HELLO WORLD CSS HEAD
====================


DESCRIPTION
-----------

##### Goal
The goal of this project is to show how use CSS in HEAD section in HTML. 
Order of CSS (from most to less important):
- element;
- head;
- file.

##### Details
Text "Hello World" in color red is displayed in browser.
